# Como executar nosso projeto

# 1º Passo

Se você clonou diretamente do GitHub, pode pular esse passo!
Pegue o arquivo sentence_embedder_valor_hora.pkl e adicione na pasta models (\artificial-intelligence\models), ele é essencial para o funcionamento do app.py

# 2º Passo

Crie o ambiente virtual com o comando `python -m venv venv`

# 3º Passo

Logo após criar o ambiente virtual (venv) utilize o comando `venv\Scripts\activate` se estiver no Windows ou `source venv/bin/activate` se estiver no Linux/MacOS

# 4º Passo

Instale as dependências do projeto (pode demorar um pouquinho) você pode ver elas no arquivo requirements.txt ou, se quiser um pouco mais de praticidade, pode executar o comando `pip install -r requirements.txt` dentro do venv

# 5º Passo

Execute a aplicação através do comando `python app.py`
